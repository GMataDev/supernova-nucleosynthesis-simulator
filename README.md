# Supernova Nucleosynthesis Simulator

Este proyecto es un simulador computacional desarrollado para la **XXXIV Feria de las Ciencias de la UNAM**, donde ganó el **Segundo lugar** en la Modalidad de **Física**. 

## Descripción
El programa modela la explosión de una supernova y la posterior creación de elementos químicos (nucleosíntesis). 

## Contenido
* `src/`: Código fuente en Python.
* `supernova_research.pdf`: Investigación teórica completa que sustenta el simulador.
* `poster_feria.pdf`: Poster presentado en la Feria de las Ciencias.
* `demo.mp4`: Video demostrativo del simulador.
* `index.html`: Código fuente de la página web del Proyecto.

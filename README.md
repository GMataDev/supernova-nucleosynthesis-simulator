# Supernova Nucleosynthesis Simulator

Este proyecto es un simulador computacional desarrollado para la **XXXIV Feria de las Ciencias de la UNAM**. 

## Descripción
El programa modela la explosión de una supernova y la posterior creación de elementos químicos (nucleosíntesis). 

## Contenido
* `src/`: Código fuente en Python.
* `Research_Paper_Supernovas.pdf`: Investigación teórica completa que sustenta el simulador.

## Futuras mejoras
